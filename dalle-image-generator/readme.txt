Generate featured images for your posts automatically using OpenAI's DALL-E.

== Description ==
This plugin allows you to automatically generate featured images for posts that don't have them using OpenAI's DALL-E API.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/dalle-image-generator`
2. Activate the plugin
3. Enter your OpenAI API key in the settings page
4. Start generating images!
